- Click **Cash Locker** tab

  - Common Pool: Split amount equally among all members
- **View Balance**:
  - Total Deposited
  - Total Withdrawn (locker payments only)
  - Remaining Balance
- **Contributions**: See each person's deposit percentage
- **Withdrawal History**: All locker payments listed

### 6. Budget Management
- Click **Budgets** tab
- Set budget for each person
- Track spending vs budget
- Visual indicators:
  - Green: Under 80% of budget
  - Yellow: 80-100% of budget
  - Red: Over budget

## Key Features

### Multi-Group Management
- **Create Multiple Groups**: Manage expenses for different trips, households, or events
- **Switch Seamlessly**: Click the group selector to switch between groups instantly
- **Independent Data**: Each group has its own members, expenses, and cash locker
- **Persistent Storage**: All groups are saved automatically in your browser
- **Easy Cleanup**: Delete groups you no longer need

### Payment Type Flexibility
Every expense supports multiple payment options:
- **Locker Payment**: Withdrawn from shared cash pool
- **Personal Payment**: Paid by a single member
- **Multiple Payers**: Split payment between multiple members and/or locker
- **Custom Splits**: Divide expenses equally, by ratio, manually, or by shares

### Real-Time Updates
When you toggle a transaction's payment type:
- Dashboard updates locker vs personal totals
- Cash Locker recalculates withdrawals
- All charts and summaries refresh automatically

### Filters & Search
In Transaction Manager:
- Search by description or payer name
- Filter by category
- Filter by payment type (all/locker/personal)

## Tips

1. **Organize by Purpose**: Create separate groups for different trips, households, or events
2. **Start with Deposits**: Add money to the cash locker before marking expenses as locker payments
3. **Track Balance**: Keep an eye on the locker's remaining balance
4. **Set Budgets**: Help members stay within spending limits
5. **Export & Backup**: Export individual groups to save backups of your data
6. **Import Anytime**: Import previously exported groups to restore or share data

## Example Workflow

### Scenario: Managing a Trip
1. Create a new group called "Bali Trip 2024"
2. Add all travelers as members with their budgets
3. Go to Cash Locker and add initial deposits
4. Add expenses as they occur during the trip
5. Monitor spending on Dashboard
6. Export the group data for records

### Scenario: Multiple Groups
1. Create "Roommates" group for household expenses
2. Create "Weekend Trip" group for a short getaway
3. Switch between groups using the group selector
4. Each group maintains its own expenses and locker
5. Export both groups for backup
